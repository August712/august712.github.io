Current Projects:

The AzureScript Project

The AzureScript Project is an initiative to create a programming language, maintained by Azure-Agst. The project started as a simple transpiler, but after talking with many friends who are familliar in the area, It's pivoted to something far greater. This initiative consists of several parts:

  • It starts with an in-development custom programming language written by myself. Specifications are TBD.
  • A supplementary compiler is also being written in C/C++, codenamed AS-Compiler. It has plans to run on most platforms using LLVM IR.

All work on this project can be seen on the project's Github Organization.

Link: https://github.com/AzureScript

Innovation Hub Mobile App

The place where I intern has a few of us working on a mobile app for their department. It's currently being made using Facebook's React Native framework. We plan on incorporating a calendar of events, a room-reservation widget, and many other features to improve quality of life at the Hub.

Link: https://github.com/FSUInnovationHub/Hub-App

SpellBook

SpellBook is a work in progress React/Node.JS/MongoDB web-app, built to aid players of Proletariat's Spellbreak. Currently it sports a loadout-sharing feature and has plans to add an entire encyclopedia of in game features.

Links: https://github.com/Azure-Agst/SpellBook, https://spellbook.azureagst.pw/
