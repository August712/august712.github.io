Steps to Reproduce:

	primary exploit: (installs menuhax)
	1.) get firmware version of device in system settings. must be 11.2.0-35u. if not, system update while 11.2 is current up-to-date version. eject sd card and plug into computer.
	2.) download tools listed above. place the `3ds/` folder and `boot.3dsx` from the starter pack on the root of the sd card. place `soundhax-usa-o3ds.m4a` on the root of your sd card. place `POST5_U_26624_usa_9221.bin` on the root of your sd card and rename it to `otherapp.bin`. merge the `/3ds` folders from `save_manager.7z`, `ctr-httpwn_v1.2.zip`, and `menuhax_v3.2.zip` with the one on your sd card. create a folder called `saveDataBackup` on the root of your sd card.
	3.) eject sd card and place back into 3ds. boot up and launch music app. play the song in `/SDCARD` titled "<3 nedwill 2016". 
	4.) make sure device is connected to internet. launch `ctr-httpwn v1.2`. press 'a' and let it run. once done, press 'start' to exit. launch `menuhax_manager v3.2`, select `install` and follow the instructions.
	5.) if the app closes, relaunch. select `configure menuhax`. Press 'a'. Select `type 1`. then, while holding the left shoulder button, touch the bottom screen. config should save successfully. press 'a' to continue. press 'b' then 'start' to exit.
	6.) in the homebrew menu, press 'start' and then 'a' to reboot.
	7.) while holding the left shoulder button, scroll over the face raiders icon. menuhax should enable and boot you into the homebrew launcher (AKA 'hbl').

	pkhex:
	8.) run `save_manager`. select game using left/right on the d-pad and press 'a' to select.
	9.) press 'a' to backup. press 'start' to exit app. turn off 3ds and remove sd card.
	10.) make changes as preferred. then save to sd card, eject, and insert to 3ds.
	11.) use menuhax to boot to hbl, then run `save_manager`. select game using left/right on the d-pad and press 'a' to select.
	12.) press '2' to restore to modified backup. press 'start' to exit app. reboot device, and launch game
	13.) changes should now reflect in game.
